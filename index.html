Comandos Minecraft

/ban (usuario) (razon)

Este comando bloquea a usuarios.

/tban (usuario) (tiempo) (razon)

Este comando bloquea a usuarios temporalmente.

/mute (usuario) (tiempo) (razon)

Para silenciar usuarios.

/staff o /mod

Aparecerá un inventario con el que puedes moderar.

/freeze (usuario)

Para congelar a un usuario para que no se mueva.

/tp (usuario) (usuario)

Para teletransportarte a usuarios instantáneamente.

/vanish

Con este comando puedes hacerte invisible.

/invsee (usuario)

Sirve para mirar y modificar inventarios.
